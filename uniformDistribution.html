Uniform Distribution

Statistics

For a random variable x that is uniformly distributed between a minimum value "a" and a maximum value "b", the uniform distribution formula is given by:

f(x) = 1 / (b - a), for a ≤ x ≤ b

Example: This could represent rolling a fair die, where each number between 1 and 6 has an equal chance of occurring.
